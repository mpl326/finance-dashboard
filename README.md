Earnings Call Sentiment & Event Study Dashboard

This project analyzes earnings call sentiment (management remarks + Q&A) and links it to stock performance using an event study methodology.
It includes a full Python/Colab pipeline for generating event-study abnormal returns (CAR) and a deployed Streamlit dashboard for interactive exploration.

Running the Dashboard Locally

If you'd like to run the dashboard on your own machine:

cd finance-dashboard
pip install streamlit pandas altair
streamlit run Dashboard_app.py
